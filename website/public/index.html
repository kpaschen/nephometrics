SRE for hire

My company offers assistance in planning, creating and maintaining cloud computing setups.

Below is a (non-exhaustive) list of things I can do for you. If you want to know more, please get in touch.

DevOps
  • Configure, extend, or troubleshoot your CI/CD pipeline
  • Automate your infrastructure setup (for example with Terraform, Ansible)
  • Set up monitoring and alerting (for example using Prometheus with Grafana and Alertmanager, integrating with tools such as PagerDuty or OpsGenie).
  • Clean up and document kubernetes configuration; adding (or removing) configuration tools such as kustomize, sonnet, or helm.
Cost Management
  • Review and explain your cloud bill; configure using custom labels, exports, queries.
  • Identify and implement savings opportunities.
  • Assist in estimating the cost of features in the design phase.
Architecture
  • Scalability review: how far will your current architecture scale, where are the bottlenecks?
  • Troubleshoot performance issues
  • Reliability assessment: could your system be more resilient? Could maintaining it take less of your engineers' time?

My experience is primarily with Google Cloud and AWS, as well as with Kubernetes environments on other providers. I can assist you with data pipelines (MapReduce, Hadoop) and messaging systems (such as Apache Kafka) as well as data storage (SQL or NoSQL).
