Politique de confidentialité

Mon site est généré statiquement par Hugo et n’enregistre pas de cookies. Je n’utilise ni de plugins social media ni d’avertissements. Par conséquent, je ne recueille pas de données personelles.

Hosting Provider

Le fournisseur de mon site internet recueille et stocke automatiquement les données et informations générales dans les fichiers journaux du serveur, lesquels sont automatiquement transmis par le navigateur que vous utilisez.

Ces données ainsi que tout mon site internet sont stockés chez mon fournisseur

METANET AG, Hardstrasse 235, CH-8005 Zürich,

dont vous trouvez ici la politique de confidentialité.

YouTube

Je peux insérer des videos de YouTube dans mes pages. En ce cas, lors de votre visite, une connection avec un server YouTube sera établie. Cette connection transmet à YouTube laquelle de mes pages vous avez visitée. Si vous êtes connectés à votre conte Google, votre identité sera transmise également. Vous pouvez éviter ça en vous déconnectant de votre conte Google.

Voir aussi la politique de confidentialité de YouTube.
